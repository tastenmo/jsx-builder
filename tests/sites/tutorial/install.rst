
Prerequisites
-------------------

Before you begin the installation process, ensure that you have the following prerequisites in place:
- Python 3.6 or higher installed on your system
- pip (Python package installer) available

Installing Sphinx
----------------------------

You can install Sphinx using pip. Open your terminal or command prompt and run the following command:
.. code-block:: bash

   pip install sphinx

Verifying the Installation
--------------------------------

To verify that Sphinx has been installed correctly, you can check the version by running:

.. code-block:: bash

   sphinx-build --version

   
This should display the installed version of Sphinx, confirming that the installation was successful.
